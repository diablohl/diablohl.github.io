手把手教你如何在自己的应用程序中导入第三方库Zxing实现扫码功能

发表于 2017-04-21 | 分类于 技术贴 | | 阅读次数

扫描二维码近两年变得非常流行，可以说无处不在，最近在学习Android时想在自己的应用中添加扫描二维码的功能，于是上网搜了搜，发现了一个功能强大的第三方开源库 Zxing

ZXing是一个开源Java类库用于解析多种格式的1D/2D条形码。目标是能够对QR编码、Data Matrix、UPC的1D条形码进行解码。其提供了多种平台下的客户端包括：J2ME、J2SE和Android。

Zxing github地址：https://github.com/zxing/zxing

网上搜了很多的教程要么不详细要么有问题，搞了很久终于搞定，于是把过程整理下来，留做备用，也分享给大家。

Zxing 是一个功能非常丰富强大的第三方库，支持多平台运行，所以源代码非常的大，由于我们只需要在Android平台上运行，所以只需要其中一小部分的功能，而网上已经有大神给我们整理好了抽取出来的来源代码，我们直接拿来用就好了，这里是抽取源代码的下载链接：百度云 密码: rkkb

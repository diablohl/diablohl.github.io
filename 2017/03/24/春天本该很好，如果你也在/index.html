春天本该很好，如果你也在

发表于 2017-03-24 | 分类于 心情随笔 | | 阅读次数

最近又降温了，三月末的天气依然还是很冷，你那里呢

翻看微信聊天记录的时候发现和你最后一条记录时间是2月24号，巧了，到今天正好是一个月的时间了

好快啊，都一个月了

回想在一起时，哪怕你半天不理我我都会急得要死然后各种找你，连环call，那时看来要是一个月不联系简直是可怕的事情，但还是这样过来了

这一个月，一个人的生活，比想象中的难熬，但也没有想象中的那么难熬

重新回归到自己的生活，开始把目光，注意力放到周围的事物上，竟也有些许不一样的体验，

只有先好好爱自己，才能给别人爱

我开始体会到这句话的真正含义了，重新注意身边的人，开始更多的联系以前的朋友，发现谈恋爱确实某些程度上来说让你错过了很多东西，恋爱之外还有很多难能可贵的东西，人不是为谈恋爱而活的

这一个多月的一个人的生活，竟然感觉到格外的轻松

再也不用心里时时刻刻挂念某个人，不用频繁的点亮手机想着去发消息，不用为对方的事情烦恼生气，不再因为没有结果的等待而失落

也不用再为了两个人见面每个月都精心计算着生活费，不用为了带她去更远更好玩的地方吃更好的东西而吃一个月的泡面，可以有时间去发现寻找去做自己的事喜欢的事，可以去买自己喜欢却一直腾不出预算的东西，最重要的是我失去了一个可以让我撕心裂肺的重要的人

放弃了之后才体会到“你走了正好，不然总担心你会走”

其实谈恋爱很累，我真的很累

你总说，你太累了，要放手了，我总是在挽留的那一个，可我也好累啊，我总是在想，再坚持一下吧，不要这么轻易放弃这么久的感情，下一次就不会这样了，可我后来发现也并不是这样，下一次似乎并不会越来越好，下一次只会是你再一次的分手

我很害怕，我不知道我该怎么办，我害怕当某个未来的分手时我不再有勇气和耐心再向你伸出挽回的手

所以还是分开吧，更何况当她的出现后就更不可能回的去了
你总说我不信任你了，可我给了你百分百的信任以后呢，你把他踩在了脚底，扔在了脑后

这件事会一直在我们之间隔应着，我也没办法百分百再相信你了，结果就是导致矛盾，矛盾之后就是你再次分手，这几乎是可以预见到的事

我还记得很早很早前某天晚上，你突然很不开心，我追问之下你告诉我，你在楼梯处看到了一个人，突然想起了他，不仅是想起了他这个人，还想起了他曾经有多么好，你说你痛恨自己的三心二意，你说你不能接受这样的自己，然后我一整晚都在安慰你，跟你说了很多 其实我那时的心态我觉得应该是介意的，可是事实我一点都不介意，因为我喜欢你这样的坦诚，内心深处的想法都和我说，不管好的坏的，那时的我才是真正住在你心里的，所以我反而很欣慰，我相信并且会做得更好，对你更好

可后来呢，不过一年多的时间，为什么一个人可以变化这么大，我不知道你瞒着我去找她时心里是怎么想的，为什么你不仅三心二意还觉得问心无愧

有那么一个人，花光了我二十年所有的爱情和热情

很难再爱了，伤的太深，特别是天蝎，受伤后只会更好的封闭保护自己，不知道以后还会不会再遇到一个能心动的人，真的太难

有句话说得好，最痛苦的不是羡慕别人，最痛苦是的羡慕曾经的自己。

春天会很好，我一个人也会很好

虽然我知道并不会有人打赏，但我就是要挂着

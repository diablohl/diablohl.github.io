没事瞎更新系列——推荐一个滚屏截长图的工具FSCapture

发表于 2017-02-26 | 分类于 没事瞎更新 | | 阅读次数

相信很多人都会有同样的需求就是在浏览文档或者网页时，需要保存长度超过当前屏幕显示的内容，也就是需要滚动屏幕浏览完的内容。如果是普通的截图工具就只能选择分多次来截图保存，这就显得很不方便，所以在这里分享一个好用的小工具FSCapture

打开文件夹后，点此启动软件

在软件界面点击红色箭头指向图标

然后点击需要截图的窗口，左键单击开始截图，按esc来停止，然后保存截图就可以了，当然在保存之前你也可以对图片先进行简单的编辑。

当然这个软件还有录屏等其他功能，如有需要可以自己去体验
